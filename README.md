# Channel MBTI

채널 엔지니어팀의 MBTI 기반 매칭 서비스

## 🎯 주요 기능

### 📊 대시보드
- 전체/팀별 MBTI 분포 통계
- Top 5 MBTI 순위
- 실시간 차트 시각화

### 🤝 매칭 & 커피챗
- 이름으로 프로필 검색
- MBTI 기반 3명 매칭 추천
  - 🤝 **Bestie Match**: 가장 궁합이 잘 맞는 찰떡 파트너
  - ⚖️ **Balancer**: 나와 다른 강점으로 균형을 맞춰주는 동료
  - 🎲 **Wild Card**: 예상치 못한 시너지를 만들어낼 매칭
- 구글 캘린더 연동으로 바로 커피챗 일정 생성
